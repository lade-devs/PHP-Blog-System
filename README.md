## Installation
Clone or download the zip
Edit include/config.php file;

 define('db_host', 'localhost');
 define('db_user', 'root');
 define('db_pass', '');
 define('db_name', 'blog');
 
 make changes to the db_user, db_pass, db_name to suit your database parameters
